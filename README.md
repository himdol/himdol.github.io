# GitHub Pages React 앱

이 프로젝트는 React와 TypeScript를 사용하여 만든 웹 애플리케이션입니다. GitHub Pages를 통해 배포됩니다.

## 🚀 배포된 사이트

[https://himdol.github.io](https://himdol.github.io)

## 🛠️ 기술 스택

- **React 19** - 사용자 인터페이스 라이브러리
- **TypeScript** - 정적 타입 검사
- **GitHub Pages** - 정적 사이트 호스팅
- **gh-pages** - GitHub Pages 배포 도구

## 📦 설치 및 실행

### 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start
```

### 빌드 및 배포

```bash
# 프로덕션 빌드
npm run build

# GitHub Pages에 배포
npm run deploy
```

## 🎯 주요 기능

- 반응형 디자인
- 모던한 UI/UX
- TypeScript 지원
- GitHub Pages 자동 배포

## 📁 프로젝트 구조

```
github.io/
├── public/          # 정적 파일
├── src/             # 소스 코드
│   ├── App.tsx      # 메인 컴포넌트
│   ├── App.css      # 스타일
│   └── index.tsx    # 진입점
├── package.json     # 프로젝트 설정
└── README.md        # 프로젝트 문서
```

## 🔧 스크립트

- `npm start` - 개발 서버 실행 (http://localhost:3000)
- `npm run build` - 프로덕션용 빌드
- `npm run test` - 테스트 실행
- `npm run deploy` - GitHub Pages에 배포
